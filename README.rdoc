==Cookbook

This is a simple cookbook application we will build during the ECRuby Chippewa Valley Ruby Camp.

TO use:

  git clone git://github.com/ecruby/cookbook.git
  cd cookbook
  rake db:schema:load
  ruby script/server
  
Requires
  Ruby
  Rails 2.3.5
  SQlite3
  

View at http://ecrubycookbook.heroku.com
